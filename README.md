# Losty Engine

A voxel game engine developed by a small indie team. Aiming for complex material dynamics and immersive gameplay.

## Branches
*   `main`: Stable releases.
*   `dev`: Main development branch.
*   `feature/*`: Feature-specific branches.
*   `bugfix/*`: Bug-specific branches.
*   `losty-game`: Experimental branch for unusual engine behaviors.
